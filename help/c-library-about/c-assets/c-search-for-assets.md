# 자산 검색{#search-for-assets}

Adobe Experience Cloud 라이브러리에서 자산을 검색하여 조직 또는 특정 폴더에서 자산을 찾을 수 있습니다.

검색 막대를 사용하여 Experience Cloud 라이브러리에서 자산을 검색합니다. 검색은 파일 이름을 검색합니다.
